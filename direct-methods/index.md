# Appendix

This appendix collects supplementary material that supports the main chapters
but is not part of the core lecture sequence.

- **Gaussian Elimination**: the classical algorithm for reducing a linear system
  to triangular form.
- **LU Decomposition**: organizing Gaussian elimination as a matrix
  factorization $PA = LU$.
- **Pivoting**: row swaps for numerical stability in Gaussian elimination and LU.
